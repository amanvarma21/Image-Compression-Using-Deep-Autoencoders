This project explored how autoencoders can be used for image compression, using the MNIST 
dataset of handwritten digits. A simple neural network was built and trained to compress 28×28 
pixel images into 32-dimensional vectors and then reconstruct them. The results showed that the 
model could retain most of the visual features of the digits, even after heavy compression. 
While the model worked well for basic compression, it had its limitations. Some reconstructions 
lacked sharpness, and the model wasn’t tested on more complex image data. Still, the project 
proved that even basic autoencoders can learn meaningful image representations without any 
manual feature engineering. 
An important takeaway was how easily a neural network could learn to compress images with 
minimal supervision. It was also a great way to understand the practical side of model training, 
tuning, and visualizing results—beyond just theory.
